こやランチャー
==============

# バージョン
1.0

# 概要
Civ4のMODとか起動するやつ。

# 動作要件
.Net Framework 4.6.2

# インストール
zipを解凍したらそのまま使えます。

# アンインストール
フォルダごと手動で削除します。
AppData\Local\coyote\ に設定ファイルが保存されます。
2KB程度ですが、気になる場合はそちらも消してください。

# つかいかた
なんとなくわかると思います。
Civ4のインストール先がデフォルトではない場合は、
ツールはCiv4を見つけることができません。
設定からインストール先を教えてあげてください。
